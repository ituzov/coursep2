# Simple Landing Page

Простой лендинг, сверстанный на HTML и CSS.

## О проекте

Проект представляет собой статическую страницу без сборщика, фреймворков и серверной части. Его можно открыть напрямую в браузере или запустить через любой локальный статический сервер.

## Структура проекта

```text
.
├── index.html
├── css/
│   └── style.css
└── README.md
```

## Как запустить

Откройте файл `index.html` в браузере.

Также можно запустить локальный сервер из папки проекта:

```bash
python3 -m http.server 8000
```

После этого страница будет доступна по адресу:

```text
http://localhost:8000
```

## Технологии

- HTML5
- CSS3

## Разработка

Основная разметка находится в `index.html`, стили - в `css/style.css`.

Рекомендуемый порядок работы:

1. Добавить или изменить секции в `index.html`.
2. Настроить внешний вид в `css/style.css`.
3. Проверить страницу в браузере на desktop и mobile ширине.

## Цель

Сделать простой, понятный и адаптивный лендинг, который можно использовать как основу для небольшой презентационной страницы.
